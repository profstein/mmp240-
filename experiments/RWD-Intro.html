RWD Big Ideas

Flexibility (stretchy)

Widths and sizes can change with context.

Breakpoints: Layout Change

At different screen sizes (widths mostly) the layout changes

RWD is …

  • Flexible Grid
  • Flexible Media (images, video and such)
  • Media Queries

A Flexible Grid Means:

setting widths in percent


.feature{
     width:40%;   
}

How do you think in %?

Think in px then translate with the magic formula *

target ÷ context = result

* no real magic involved

  • Target: The number you want in pixels
  • Context: The width of the containing element
  • Result: The number as a percent. Move the decimal two places to the right

Example

Target width of my sidebar is 200px. It is in the body which is set to 960px.

200 ÷ 960 = .2083333


.sidebar{
     width:20.83333%;   
}

Padding and Margin

Same formula but..

  • Margin: the context is the width of the container
  • Padding: the context is the width of the element itself
