Style The Text

  • Make sure the most important items stand out from the rest.
  • Each different kind of content needs a different type treatment (color, font, size etc).
  • The same kinds of content should get the same type treatment.

CSS Typography Properties

  • Font: font-family
  • Weight: font-weight
  • Size: font-size
  • Style (italic): font-style
  • Line Height: line-height
  • Color: color
  • Background Color: background-color
  • Alignment: text-align
  • Case: text-transform
