About the Class

The Big Picture

What we will do

You will learn to make standards-based,
visually attractive, usable
multi-screen websites that
communicate a clear message

Main Learning Objectives

you will learn to:

Plan

Design

Build

Iterate

Deploy

Plan

Researching, creating planning documents like Creative Brief, Sitemap, Wireframes

Design

Design Theory for the Web

  • Basic Design Principles
  • Page Layout Principles
  • Gestalt Theory
  • User Experience (UX)

Design Specifics

  • Typographic Hierarchy
  • Layout
  • Styleguide
  • Integrating Interactivity

Build

Actually writing the HTML, CSS and JavaScript for the site and creating/editing media

  • HTML (valid, semantic)
  • CSS (valid, organized)
  • JavaScript (not cvered)

Web Design Tools

  • Editor: we will use Brackets
  • CSS Preprocessor: Sass &
  • Version Control: Git and GitHub

Iterate

This is the process of getting and receiveing feedback, testing and doing the design and build over again based on the outcomes.

Deploy

Actually putting your site on the web so it is accessible. We will also focus on performance.

Overarching Concepts

  • User Experience (UX)
  • WorkFlow
  • Reponsive Web Design
